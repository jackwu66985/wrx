# 微信小程序：个人简介 + 毕业设计全案 + 其他作品

## 功能结构

- **简介页**：头像、核心信息、技能标签、个人陈述
- **毕业设计页**：项目背景、调研、设计目标、流程、成果与复盘
- **作品集页**：不同类别的个人作品卡片展示
- **联系页**：联系方式、可提供服务、合作方式

## 快速使用

1. 打开微信开发者工具
2. 导入当前目录 `/workspace/wrx`
3. 使用测试号（`touristappid`）预览
4. 根据你的信息替换以下文件中的示例数据：
   - `pages/home/home.js`
   - `pages/thesis/thesis.js`
   - `pages/works/works.js`
   - `pages/about/about.js`
